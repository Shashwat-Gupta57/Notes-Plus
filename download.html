How to install

1

Allow installs

Open Settings → Security → enable "Unknown sources" or allow your browser to install apps.

2

Download APK

Tap the download button above. The file will save to your Downloads folder.

3

Install

Open the APK file and tap Install. Android may ask for confirmation — that's normal.

4

Done!

Open Notes+ from your app drawer. The secret vault is already there — you just need to find it.

Why sideload?

Notes+ is a personal project — not a commercial app. It's not on the Play Store because it doesn't need to be. There are no ads, no analytics, no trackers, and no servers collecting your data. The APK is hosted on GitHub, fully open for inspection. You install it directly because that's the simplest, most honest way to distribute software.
